============================================================
 안전점검 모니터링 시스템 - 설치 및 사용 안내
============================================================

이 폴더는 "안전점검 수행기관 지정공고"를 여러 기관 게시판에서
자동으로 수집·선별해 주는 프로그램입니다.

폴더 구성:
  안전점검_모니터링.exe   ← 실행 파일 (더블클릭으로 실행)
  _internal\               ← 프로그램 내부 파일 (삭제하지 마세요)
  install.bat              ← 최초 1회 설치 스크립트
  .env                     ← API 키 설정 파일
  config\sites.xlsx        ← 점검할 사이트 목록
  config\keywords.json     ← 키워드 필터 설정
  data\  logs\  results\   ← 실행 결과가 저장되는 폴더

------------------------------------------------------------
 1. 최초 설치 — install.bat 실행 (playwright 설치)
------------------------------------------------------------
일부 자바스크립트 기반 게시판 수집을 위해 playwright(브라우저
자동화)가 필요합니다.

  1) install.bat 파일을 마우스 오른쪽 클릭
  2) "관리자 권한으로 실행" 선택 (권장)
  3) 검은 창에 다음이 자동으로 진행됩니다:
       - pip install playwright
       - playwright install chromium  (크롬 엔진 다운로드)
  4) "설치 완료!" 메시지가 보이면 아무 키나 눌러 창을 닫습니다.

  ※ 인터넷 연결이 필요하며, 최초 1회만 실행하면 됩니다.
  ※ Python 이 설치되어 있지 않다면 install.bat 실행 전
     https://www.python.org 에서 Python 3.12 이상을 먼저
     설치하세요. (설치 시 "Add Python to PATH" 체크)

------------------------------------------------------------
 2. .env 파일 — API 키 설정
------------------------------------------------------------
AI 선별 기능(Claude / Gemini)을 사용하려면 API 키가 필요합니다.
키가 없어도 프로그램은 동작하며, 이 경우 "키워드 기반 폴백"으로
자동 전환됩니다(정확도는 다소 낮아집니다).

  1) 메모장으로 .env 파일을 엽니다.
     (.env 가 없으면 새로 만들어 아래 내용을 입력하세요)

  2) 아래 형식으로 키를 입력합니다 (= 뒤에 본인 키 붙여넣기):

       ANTHROPIC_API_KEY=sk-ant-여기에_클로드_키_입력
       GEMINI_API_KEY=여기에_제미나이_키_입력
       AI_PROVIDER=gemini
       CRAWL_DELAY=2
       MAX_PAGES=3
       LOG_LEVEL=INFO

     - AI_PROVIDER : claude 또는 gemini 중 우선 사용할 AI 선택
                     (선택한 쪽 키가 없으면 반대쪽을 자동 시도)
     - 둘 중 하나만 있어도 동작합니다.
     - CRAWL_DELAY : 사이트 간 대기 시간(초). 너무 짧으면 차단될 수
                     있으니 2~3 을 권장합니다.
     - MAX_PAGES   : 게시판당 수집할 페이지 수.

  3) (선택) 매일 정해진 시각 자동 실행을 원하면 아래 두 줄 추가:

       SCHEDULE_ENABLED=Y
       SCHEDULE_TIME=09:00

  4) 저장(Ctrl+S) 후 메모장을 닫습니다.

  ※ API 키 발급처
     - Claude  : https://console.anthropic.com  (키 형식: sk-ant-...)
     - Gemini  : https://aistudio.google.com/apikey

  ※ 주의: .env 의 API 키는 외부에 공유하지 마세요.

------------------------------------------------------------
 3. sites.xlsx — 점검할 사이트 목록 수정
------------------------------------------------------------
config\sites.xlsx 를 엑셀로 열어 점검 대상을 관리합니다.

  컬럼 구성 (A열부터):
    A) 기관명        예) 서울특별시
    B) 사이트유형
    C) 게시판명      예) 고시공고
    D) 게시판URL     수집할 게시판 목록 페이지 주소
    E) 페이지파라미터 예) page  (페이지 넘김에 쓰이는 변수명)
    F) 활성화        Y = 수집함 / 그 외 = 건너뜀
    G) 비고          (자동) 연속 실패 시 "점검필요" 표시됨

  수정 방법:
    1) 새 기관을 추가하려면 빈 행에 기관명/게시판명/URL 등을 입력하고
       F열(활성화)에 Y 를 입력합니다.
    2) 특정 사이트를 잠시 제외하려면 F열을 비우거나 N 으로 바꿉니다.
    3) URL 이 바뀌었다면 D열을 최신 주소로 갱신합니다.
    4) 저장 후 엑셀을 반드시 닫습니다.
       (열려 있으면 프로그램이 비고란을 저장하지 못합니다)

  ※ 첫 행(1행)은 제목 행이므로 지우지 마세요.
  ※ 게시판URL 을 모르는 신규 사이트는 프로그램의 "게시판 등록"
     기능으로 자동 탐색해 채울 수 있습니다.

------------------------------------------------------------
 4. 실행 방법
------------------------------------------------------------
  1) 안전점검_모니터링.exe 를 더블클릭합니다.
  2) 프로그램 창이 열리면:
       - [크롤링 실행]  : 활성화(Y)된 모든 사이트 수집
       - [선택 실행]    : 왼쪽 목록에서 체크한 사이트만 수집
       - [중지]         : 진행 중 안전하게 멈춤
       - [게시판 등록]  : 신규 사이트의 게시판 URL 자동 탐색
       - [설정]         : .env 값을 GUI 에서 변경
  3) 수집 결과는 오른쪽 표에 표시됩니다.
       - 노란색 행 = 신규 공고
       - 행을 더블클릭하면 해당 게시글이 브라우저로 열립니다.
       - [엑셀 저장] 으로 results\ 폴더에 .xlsx 로 저장됩니다.

  결과/이력 저장 위치:
    - results\  : 엑셀 결과 파일
    - data\     : 신규/기존 판별용 이력(history.json)
    - logs\     : 스케줄 실행 로그 등

------------------------------------------------------------
 자주 묻는 질문 / 문제 해결
------------------------------------------------------------
  Q. 실행하면 바로 꺼져요.
     A. config\sites.xlsx 와 _internal\ 폴더가 EXE 와 같은 폴더에
        있는지 확인하세요. 폴더 구조를 옮기지 마세요.

  Q. "AI 폴백 모드" 라고 나와요.
     A. .env 의 API 키가 없거나 잘못되었습니다. 2번 항목을 확인하세요.
        (키워드 기반으로는 계속 동작합니다)

  Q. 특정 사이트가 "점검필요" 로 표시돼요.
     A. 연속 3회 수집 실패한 사이트입니다. 해당 게시판 URL 이
        바뀌었는지 확인하고 sites.xlsx 의 D열을 갱신하세요.

  Q. 엑셀 저장이 안 돼요.
     A. 저장하려는 엑셀 파일이 이미 열려 있으면 저장되지 않습니다.
        파일을 닫고 다시 시도하세요.
